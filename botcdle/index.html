Rules

Try and guess the correct BOTC character. A new one becomes available each day (local timezone). After each guess, you will learn how correct you are for all columns: Green = fully correct, yellow = partially correct, Red = wrong.

Ability Type categorizes a characters ability into one or more out of 8 categories: Start knowing info, Pick player(s) at night, Pick characters(s) at night, Learn info at night, Daytime ability, Passive, Setup and Victory Condition.

An ability that triggers on death is considered passive. Info gained from picking a player at night does not count as Learn info at night, as it's considered part of the picking. Abilities that allow a character to gain abilities from other characters only consider the original ability.

The First night wakeup and Other night wakeup columns gives you a hint of how your character wakes up compared to the target. If the cell says 'Too early', it means your guess wakes up too early compared to the target. If it says 'Not comparable', it means either your guess or the target does not wake up (or wakes up at an unpredictable time). Characters of the same type that wakes up next to eachother, and where the relative order has no effect on the game are considered to wake up at the same time. This includes demons (despite the fact that their kills would be affected by the order). Demon and minion info is not considered.

A character not waking up at all, and a character that wakes up unpredictably are not considered equal.
